1. Структура страницы

	header -> main -> footer

2. Страницы должны иметь конкретные имена. Например, страница с галереей должна называтся gallery.

3. Для каждой страницы должен быть свой css файл, и называтся именем страницы.

4. Для повторяющихся блоков можно создавать свой стилевой файл.

5. Стили хранятся в отдельной папке с названием Styles.

6. Папка с картинками распологается внутри Styles.

7. Скрипты JavaScript должны хранится в папке js в корне.
